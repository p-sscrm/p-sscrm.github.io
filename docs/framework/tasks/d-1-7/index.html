D.1.7 Establish Response Partnerships

Task Details

Objective

Develop external incident information sources through collaboration.

Definition

Partners can help during vulnerability response by providing information about compromises, enabling improved response efforts, and collaborating in dealing with attacks. Adversaries are only empowered when no collaboration exists between software organizations.

Assessment Questions

  1. What vulnerability information and incident response relationships do you have with external organizations?
