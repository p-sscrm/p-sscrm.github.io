E.3.x Development Environment Scanning Tools

Control Details

Objective

Detect anomalies in the development infrastructure.

Definition

Automated scanning tools can detect anomalies and undesired behaviors in each environment, including development, build, testing, and deployment.

Assessment Questions

  1. What tools do you use to secure your development environment? For example, tools to remove viruses, scan files downloaded from the internet before execution, detect signatures of anomalous payloads, and identify changes in configuration settings.
